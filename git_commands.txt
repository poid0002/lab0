git clone - Скачать удаленный репозиторий на локальный компьютер
git status - Показать состояние файлов (измененные, подготовленные к коммиту)
git add [файл] - Добавить файлы в индекс для последующего коммита
git commit - Создать коммит с описанием изменений
git push - Отправить локальные коммиты на удаленный сервер
